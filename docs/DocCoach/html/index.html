CoachIMG
Basic Android application to calculate IMG and save its history
Android IMG


Application Android

Cette application mobile a été développé sous AndroidStudio en Java. Son objectif est de permettre à ses utilisateurs d’avoir accès à un calcultateur d'IMG et d'enregistrer les variables de taille, poids, age et la date dans une Base distante via une API PHP et en local avec SQLite afin de retrouver plus tard le suivi de son IMG.
